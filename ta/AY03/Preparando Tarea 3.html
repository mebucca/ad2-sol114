Distribuciones continuas — recordatorio rápido

Una variable aleatoria continua toma infinitos valores en un intervalo. Su comportamiento se describe con:

  • PDF \(f(x)\): densidad de probabilidad. No es \(\mathbb{P}(X=x)\).
  • CDF \(F(k) = \mathbb{P}(X \leq k) = \int_{-\infty}^{k} f(x)\,dx\)

Las dos distribuciones más importantes en este curso:

Distribución PDF Parámetros
Uniforme \(X \sim U(a,b)\) \(f(x) = \dfrac{1}{b-a}\) \(a\) (mín), \(b\) (máx)
Normal \(X \sim N(\mu,\sigma^2)\) \(f(x) = \dfrac{1}{\sigma\sqrt{2\pi}}\,e^{-\frac{(x-\mu)^2}{2\sigma^2}}\) \(\mu\) (media), \(\sigma\) (desv. est.)

Momentos — ¿qué son?

Los momentos resumen una distribución con un solo número:

1er momento — Valor Esperado: \[\mathbb{E}(X) = \int x\,f(x)\,dx\] 2do momento — Varianza: \[\mathbb{Var}(X) = \int \bigl(x - \mathbb{E}(X)\bigr)^2 f(x)\,dx = \mathbb{E}(X^2) - \bigl[\mathbb{E}(X)\bigr]^2\]
Uniforme \(U(a,b)\) Normal \(N(\mu,\sigma^2)\)
\(\mathbb{E}(X)\) \(\dfrac{a+b}{2}\) \(\mu\)
\(\mathbb{Var}(X)\) \(\dfrac{(b-a)^2}{12}\) \(\sigma^2\)

Propiedades operacionales

Si \(X\), \(Y\) son variables aleatorias y \(c\) constante:

\[\mathbb{E}(c) = c \qquad \mathbb{E}(X+c) = \mathbb{E}(X)+c \qquad \mathbb{E}(cX) = c\,\mathbb{E}(X)\] \[\mathbb{E}(X+Y) = \mathbb{E}(X)+\mathbb{E}(Y) \qquad \] \[\mathbb{Var}(c) = 0 \qquad \mathbb{Var}(cX) = c^2\,\mathbb{Var}(X)\] \[\mathbb{Var}(X \pm Y) = \mathbb{Var}(X)+\mathbb{Var}(Y) \qquad \text{(solo si } X \perp Y\text{)}\]
