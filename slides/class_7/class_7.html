Estandarización

(puntaje Z)

Puntaje Z


Sea \(X\) una variable aleatoria, tal que:


  • \(\mathbb{E}(X) = \mu\)

  • \(\mathbb{Var}(X) = \sigma^2\). Equivalentemente, \(\sqrt{\mathbb{Var}(X)} = \sigma\)


El puntaje Z de dicha variable se define como:


\[\begin{align} Z &= \frac{X - \mathbb{E}(X)}{\sqrt{\mathbb{Var}(X)}} \\ \\ &= \frac{X - \mu}{\sigma} \end{align}\]

Valor esperado de Z

El valor esperado de una variable Z es 0:


\[ \begin{aligned} \mathbb{E}(Z) &= \mathbb{E}\left(\frac{X - \mu}{\sigma}\right) \\ \\ &= \frac{1}{\sigma} \mathbb{E}(X - \mu) \\ \\ &= \frac{1}{\sigma} ( \mathbb{E}(X) - \mathbb{E}(\mu) ) \\ \\ &= \frac{\mu - \mu}{\sigma} = 0 \end{aligned} \]


Des-centrar una variable: cada valor individual \(x\) se desplaza por la misma cantidad, \(-\mu\) .

Varianza de Z

La varianza (y desviación standard) de una variable Z es 1:


\[\begin{aligned} \mathbb{Var}(Z) &= \mathbb{Var}\left(\frac{X - \mu}{\sigma}\right) \\ \\ &= \frac{1}{\sigma^2} \mathbb{Var}(X - \mu) \\ \\ &= \frac{1}{\sigma^2} ( \mathbb{Var}(X) - \mathbb{Var}(\mu) ) \\ \\ &= \frac{\sigma^2 - 0}{\sigma^2} = 1 \end{aligned}\]


Re-escalamiento de una variable: cada valor individual \(x\) se escala por \(\frac{1}{\sigma}\)

Propiedades variable en pje. Z


  • Sin unidad: Toda variable en puntaje Z tiene la misma media y la misma desviación estándar. Útil para comparar valores que provienen de distintas distribuciones (“estandarizar”).


  • Preserva forma: des-centramiento y re-escalamiento no cambia la forma de la distribución original.

Proceso “generador” de datos

Supongamos que de acuerdo a determinantes genéticos, la altura de un hombre de Islas Faroe y la de una mujer de Filipinas pueden describirse respectivamente como una realización de las siguientes variables aleatorias:


  • Hombre Islas Faroe

\[A_{HIF}\sim \text{Normal}(\mu = 179,\sigma = 5)\]


  • Mujer Filipinas

\[A_{MF}\sim \text{Normal}(\mu = 150,\sigma = 7)\]

Altura en dos “poblaciones”

mu_faroe=179; sigma_faroe=5
mu_filip=150; sigma_filip=7

hombres_faroe <- rnorm(n=53000, mean = mu_faroe, sd=sigma_faroe)
mujeres_filip <- rnorm(n=113000000, mean = mu_filip, sd=sigma_filip)

hombres_faroe_z <- (hombres_faroe - mu_faroe)/sigma_faroe
mujeres_filip_z <- (mujeres_filip - mu_filip)/sigma_filip

datos_faroe <- tibble(altura = hombres_faroe, z=hombres_faroe_z)
datos_filip <- tibble(altura = mujeres_filip, z=mujeres_filip_z)
